node

官网下载链接中的文件好像不是新的，我下到的是6.x的，而稳定版当时好像已经到8.x了，而且用官网源码下载的需要比较高版本的gcc，而升级gcc又比较复杂，所以选择在GitHub下载二进制版本来安装GitHub链接，我下载的是9.x版本，所以输入命令curl -sL https://rpm.nodesource.com/setup_9.x | bash -，这是官方的自动安装脚本，然后就安装好了，输入命令node -v以及npm -v查看版本
